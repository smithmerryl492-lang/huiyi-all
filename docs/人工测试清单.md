### API 服务

需要先在 `F:\会晓AI\services\.env` 配好共享远程 `HUIXIAO_DATABASE_URL`。API 不再允许默认连接本机 PostgreSQL，缺少该配置会直接启动失败。

```powershell
cd F:\会晓AI\services\api_server
$envFile='F:\会晓AI\services\.env'
Get-Content $envFile | ForEach-Object {
  if ($_ -match '^\s*([^#][^=]+?)\s*=\s*(.*)\s*$') {
    [Environment]::SetEnvironmentVariable($matches[1].Trim(), $matches[2].Trim().Trim('"').Trim("'"), 'Process')
  }
}
$env:HUIXIAO_ASR_SERVICE_URL='http://127.0.0.1:8081'
$env:HUIXIAO_AI_SERVICE_URL='http://127.0.0.1:8082'
python -m uvicorn app.main:app --host 127.0.0.1 --port 8080
```

- 上传文件：`POST /api/v1/files/upload`
- 处理任务：`POST /api/v1/tasks/{task_id}/process`
- 查询任务：`GET /api/v1/tasks/{task_id}`
- 查询结果：`GET /api/v1/tasks/{task_id}/result`
- 知识库问答：`POST /api/v1/knowledge/ask`

## Android App

- `staging` 默认访问测试服：`http://43.154.197.96:28080/api/v1`。当前测试服容器已关闭且暂不允许开启，只有测试负责人开放槽位后才能验收该地址。
- 本机联调时必须显式用 Gradle 参数覆盖 API 地址为 `http://10.0.2.2:8080/api/v1`，不要用 `adb reverse` 掩盖真实后端地址。
- 专用测试模拟器：`127.0.0.1:16416`，MuMu 名称为“会晓ai测试模拟器”
- 首页可创建预约会议，预约后出现在今日会议列表。
- 录音结束后进入生成页，会上传本地 m4a 到 API 服务。
- 导入文件后点击生成，会上传本地导入文件到 API 服务。
- 生成完成后进入会议详情，展示真实 ASR 转写和 LLM 纪要。
- 会议详情展示说话人列表，可进入说话人管理重命名。
- 转写原文展示每句话的来源时间位置，优先显示开始/结束时间。
- 知识库页可输入问题，调用 `POST /api/v1/knowledge/ask`，答案应展示来源片段。
